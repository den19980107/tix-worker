注意事項

  1. 旅客可預訂29天內（含乘車當日）至發車前1小時為止之車次，且每逢週五及週六至多可預訂四週後至週日之車票。(如因特殊事由調整開放時間，本公司將另行公告）
  2. 每筆交易至多可預訂10張票，不限身分，去回程至多可預訂各5張。購買孩童票之兒童乘車時，建議至少需有一位成人同行為宜。
  3. 自2023/12/01起，早鳥優惠採記名使用且僅限單程票適用，購買早鳥優惠票者應填寫每位搭乘者中華民國身分證號，外籍旅客則填寫護照號碼，所填寫之證號末5碼將列印於票面。乘車時應攜帶與票面登錄證號相符之有效證件正本以供查驗，無法出示有效證件或非本人持用者，視為無票乘車，應照章補票。詳情請參閱企網早鳥優惠說明。
  4. 為維護您的權益，請於規定時間內完成付款。如未完成付款，本系統將自動取消訂位。
  5. 座位喜好功能係依列車訂位概況媒合您期望的座位，如該車次已無合適座位時，為讓您仍可順利完成訂票作業，將由系統自動配置座位。座位喜好功能僅適用於單人乘客。
  6. 身分說明：
    • 孩童票購票說明：未滿12歲之兒童得購買孩童票，身高未滿115公分之兒童不佔位得免票，但應由已購全票或成年旅客陪同，且每位旅客攜帶之免票孩童以2人為限。身高超過150公分且未滿12歲兒童，或身高超過115公分且未滿6歲兒童應出示身分證明。
    • 敬老票購票說明：乘車當日年滿65歲(含)以上之中華民國國民，憑有效之身分證明文件得購買敬老票。
    • 愛心票購票說明：持有身心障礙證明之中華民國國民得購買愛心票。（身心障礙證明背面之「必要陪伴者優惠措施」欄位註記有「國內大眾運輸工具」字樣者，始提供必要陪伴者一人愛心票優惠）。如同行陪伴者為外籍人士，乘客資訊請以身心障礙旅客之姓名及身分證字號登錄。本系統目前並無提供預訂無障礙座位，若您有預訂無障礙座位之需求時，請致電高鐵客戶服務專線：4066-3000（苗栗、台東、金門、馬祖地區及行動電話請撥02-4066-3000。國際來話請撥：+886-2-4066-3000。以上皆為付費電話，依一般市話及行動電話、國際電話費率標準計費。服務時間：06:00~24:00）或請至各車站售票窗口購票。
    • 大學生票購票說明：中華民國大專院校在學生始得購買。持用學生票乘車時，應攜帶「學生證正本」，如學生證無當學期註冊章，須合併出示「在學證明正本」方得適用優惠。無本人有效證件者應照章補票並得加收票價差額百分之五十。詳情請參閱企網大學生優惠活動說明。
